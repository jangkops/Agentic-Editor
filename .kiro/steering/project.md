# AI Editor – Project Steering  ## Stack (STRICT) - Frontend: Electron + Vanilla JavaScript + HTML + CSS - Backend: Python 3.11+ / FastAPI / HTTPX - NO: React, TypeScript, Rust, SQLite, any JS framework  ## Data Persistence - ALL data → app.getPath('userData') subfolders ONLY - userData/settings/settings.json     → profile name, gateway config - userData/history/YYYY-MM-DD.json   → daily conversation history - userData/checkpoints/{wf_id}.json  → agent workflow state - userData/usage/usage.json          → token counts, cost tracking  ## LLM Calls - ALL calls via Bedrock Gateway ONLY - Gateway auth: appid + apitoken headers - NO direct boto3, NO Anthropic SDK, NO OpenAI SDK  ## Auth - AWS SSO only: `aws sso login` + `aws configure export-credentials` - settings.json stores profile NAME only — never credentials  ## Multi-Agent (REQUIRED) - Coordinator → Planner (Opus) → Generator (Sonnet) → Evaluator (Opus) - Max 3 iterations per workflow - Single-agent = anti-pattern, reject immediately  ## Skills in Use - webapp-testing: write Playwright test after each UI component - frontend-design: VS Code-style dark industrial aesthetic, no generic AI look
